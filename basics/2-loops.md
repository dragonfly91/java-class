# 繰り返し数

`foo()`機能が何回実行されるかを数えましょう。コードを書く必要はありません。よろしければ、ノートで計算してみてください。

```Java
for (int i = 0; i < 10; i++) {
  foo();
}
```

```Java
int j;
for (j = 0; j < 7; j++) {
  foo();
}
```

```Java
for (int i = 0; i < 10; i = i + 2) {
  foo();
}
```

```Java
for (int i = 10; i > 1; i = i - 2) {
  foo();
}
```

```Java
for (int i = 0, j = 10; i < 10 && j > 5; i++, j--) {
  foo();
}
```

```Java
for (int i = 0; i < 10;) {
  foo();
}
```

```Java
for (int i = 4; i < 20; i = 2 * i + 3) {
  foo();
}
```

```Java
for (int i = 0; i < 10; i++) {
  for (int j = 0; j < 10; j++) {
    foo();
  }
}
```
